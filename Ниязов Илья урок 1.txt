4 Задание
Man - это система справки о командах для терминала
Cd – позволяет перемещаться по каталогам
Ls – Показывает файлы и папки директории в которой мы находимся
Hostname – имя которое присваивается компьютеру, и позволяет подключиться к нему без IP
Whoami – показывает имя пользователя
Pwd - Вывести полный путь до текущей рабочей директории, в которой находится пользователь.
Ip – узнать IP своего компьютера
Sudo – поднимает наше положение до уровня root
Reboot – перезапустить компьютер
Poweroff – отключить питание
apt или apt-get – используется при установке программ

5 задание
Для того, что бы пользоваться фаерволом ufw, его надо в начале установить при помощи команды 
sudo apt-get install ufw
Что бы запустить его используется команда
sudo ufw enable
Для добавления правил используем команду
sudo ufw allow ssh
Для отключения ufw используется команда
sudo ufw disable

Что бы работать при помощи ключей делаем следующие действия:
1)	Создаем пары ключей и размещаем их  ssh-keygen -t rsa 
2)	Копируем ключ для пользователя ssh-copy-id user@12.34.56.78
На этом все, теперь мы можем входить по ssh без ключа
