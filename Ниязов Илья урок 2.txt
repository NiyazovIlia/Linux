1) Используя команду cat мы можем создавать файлы, записывать в файлы, соединять файлы, копировать файлы.
Используя команду cat > test.txt, мы создаем новый текстовый файл и записываем в него что-нибудь
При помощи команды cat test.txt, мы можем прочитать содержимое файла
Используя команду cat test.txt >> test_2.txt, мы копируем текст из первого файла и записываем во второй
Используя команду cat test_2.txt > newtest.txt, мы копируем текст из файла в другой и автоматически создаем его

2) Используя команду ls -dl .[^.]*, мы можем посмотреть исключительно скрытые файлы
Можно так же использовать ls -al, но тогда нам покажет не только скрытые но и обычные файлы

3) Используя curl -o flaskapp.py  https://raw.githubusercontent.com/mtuktarov/linux_goodies/master/docker/python/uwsgi/flaskapp.py, 
мы сохраняем нашу программу.
Так же при помощи, sudo apt install python3, sudo apt install python3-pip, pip3 install flask, скачиваем все недостающие файлы для запуска.
Если я правильно понял, я заменил host = os.getenv('FLASK_HOST', '127.0.0.1') на ip своего ubuntu, и через свой хост windows подключился через ssh
и запустил этот калькулятор.
Для запуска этой программы только по названию, используем следующие команды: 
1) sudo mv /home/ilia/flaskapp.py bin/flaskapp
2) sudo chmod +x /opt/bin/flaskapp
